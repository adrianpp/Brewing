Hardware Connections
=====================
1-wire/i2c : aviation connector wired as follows:
pin 1 - ground
pin 2 - power
pin 3 - signal

3 way valve (cr02 wiring) : aviation connector wired as follows:
pin 1 - close (blue)
pin 2 - open (brown/red)
pin 3 - ground (yellow)
